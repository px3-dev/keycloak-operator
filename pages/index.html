keycloak-operator

Helm chart for the Keycloak Kubernetes operator

Quick start

# Add the repo
helm repo add px3-dev https://px3-dev.github.io/keycloak-operator

# Install
helm install keycloak-operator px3-dev/keycloak-operator

Mirror images

helm install keycloak-operator px3-dev/keycloak-operator \
  --set image.repository=mirror.example.com/keycloak/keycloak-operator \
  --set keycloakImage.repository=mirror.example.com/keycloak/keycloak
